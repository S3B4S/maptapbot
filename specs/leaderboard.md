# Leaderboard

As an user I want to see a leaderboard overview of how people in the server that play the game are doing.

There are two types of leaderboards;
1. Daily
2. Permanent

In both cases, the leaderboard is always scoped to the current guild only.

## Daily

Usage
```
/leaderboard_daily
```

Shows a table that gives an overview of how users are doing with todays challenge only.
Sorted descendingly by total score.

## Permanent
Usage
```
/leaderboard_permanent
```

This shows a table in similar fashion, but averages the score across every column.
Sorted descendingly by total score.
