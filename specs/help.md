# `/help` command

Available to everyone, in guilds and in DMs with the bot.
The response is always ephemeral (only visible to the invoker).

## Behaviour

For regular users, lists all user-facing commands with a short description.

For admins (see [admin.md](./admin.md) for how admin status is determined), additionally lists all admin commands with a short description.

## User-facing commands (shown to everyone)

| Command | Description |
|---|---|
| `/today` | Get a link to today's maptap challenge |
| `/leaderboard_daily` | Show today's scores for this server |
| `/leaderboard_permanent` | Show the all-time average scores for this server |
| `/leaderboard_challenge_daily` | Show today's challenge scores for this server |
| `/leaderboard_challenge_permanent` | Show the all-time challenge averages for this server |
| `/help` | Show this help message |

## Admin commands (additionally shown to admins)

See [admin.md](./admin.md) for the list of admin commands.
